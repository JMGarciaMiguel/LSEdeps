An object is a nominal which is the (direct) object of a verbal predicate. 

~~~ sdparse
PRO1 LITEN-PERSON HA\*INTE PERSON^TECKEN PRO1 PU@g \n When I was small, I didn't have a name sign.
advcl(HA\*INTE, PERSON^TECKEN)
~~~
